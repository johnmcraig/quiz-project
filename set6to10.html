Question 6

Q: Name and describe 4 different kinds of positioning in CSS

A: Specifically for the position property in CSS, some are:

  1. Absolute: an element will display close to the nearest ancestor/parent element, but will stay in place to the viewport of a page. If there is not any ancestor, then the element affixes to the document body.
  2. Relative: an element will be position relative to its normal location.
  3. Static: this is the defualt position of an element.
  4. Fixed: an element will be fixed to a the viewpoet of a webpage in the browser and will scroll down the page as the viewer scrolls down.

Question 7

Q: What is floating?

A: Floating is a CSS property to manipulate an element to a specific place left or right within a webpage using div { float: left/right; }.

Question 8

Q: How do you stop something from floating?

A: To override or stop the floating property, we can use the clear property in CSS instead.

A: Web servers are computers or devices that deliver Web pages through HTTP or URL requests.
