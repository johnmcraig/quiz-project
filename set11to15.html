Question 11

Q: How do we tell JavaScript we have a block of code?

A: We use the open and closed curly braces { and } to suround a block of code, which indicates there is code to be executed within them.

Question 12

Q: What is an object?

A: An object is a collection of properties, where a property is an association between a name (or key) and a value.

Question 13

Q: What are the 4 pillars of object oriented programming?

A: The four pillars are:

  1. Polymorphism
  2. Encapsulation
  3. Abstraction
  4. Inheritance

Question 14

Q: Describe each of the 4 pillars of object oriented programming

A: The pillars are described as:

  1. Polymorphism - The process where an object or methods takes many different forms.
  2. Encapsulation - Binding variables and/or objects together with methods through the Class property.
  3. Abstraction - Only display relative information to the world and hiding all other information.
  4. Inheritance - The process of creating a new method or class that inherits (hince the word) elements of another existing method or class.

Question 15

Q: What is a function?

A: A function is a group of resuable code that can be executed when called upon.
