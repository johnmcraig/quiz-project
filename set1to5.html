Question 1

Q: What do we call the style of design that targets small devices by default and scales up to large monitors?

A: Responsive Design.

Question 2

Q: What feature of CSS do we make use of to change the design for different screen sizes?

A: We use the CSS media query: @media on screen [screen-width: #px]

Question 3

Q: What is the name of the library we use to help with our designs?

A: Bootstrap, formally known as Twitter Bootstrap.

Question 4

Q: Describe the grid system in that library

A: Bootstrap divides the HTML page into 12 sections throughout the page. These grids will colopse on one another depending on the screen size of a device used.

Each grid can be a multiple of 12, so there can be multiple grids on one line each containing their own content.

Question 5

Q: Write the HTML for a DIV that holds three other DIVs that are equal size on medium or larger screens using the grid system. Give them each a different background color and put some text in them.

We can write an example as follows: < div id="col-green" class="col col-md-4" > and with the CSS style #col-green { background-color: green; }
